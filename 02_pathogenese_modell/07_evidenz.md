## **7\. Pharmakologische Evidenz: Selbstversuch**

### **7.1 Methylphenidat (Ritalin)**

| Parameter                   | Befund                                                                                                                                                           |
| :-------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Dosis**                   | 7–14 mg (unter klinischer Schwelle für Neurotypische: 20–30 mg)                                                                                                  |
| **Wirkung auf Wahrnehmung** | Potenzierung, nicht Dämpfung. Konzentration erhöht, Diffusität der Wahrnehmung.                                                                                  |
| **Wirkung auf Migräne**     | Vollständige Migränefreiheit über Beobachtungszeitraum bei wöchentlicher Baseline.                                                                               |
| **Nebenwirkung**            | Psychotischer Rebound: instabile Hyperwahrnehmung, hoher Blutdruck, Alarmierung, Identitätsstörung. Dosisabhängig, reproduzierbar bei voller und halber Dosis.   |
| **Bewertung**               | Kontraindiziert – nicht wegen Hauptwirkung, sondern wegen Rebound. Dopamin-Hypersensitivität bestätigt. Migränesuppression über dopaminergen Mechanismus belegt. |

### **7.2 Lisdexamfetamin (Elvanse)**

| Parameter                       | Befund                                                                                                                                                                                               |
| :------------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Dosis**                       | 15 mg (halbe 30-mg-Kapsel in Lösung) → ca. 4–5 mg aktives Dexamfetamin. Ziel: Reduktion auf 7,5 mg.                                                                                                  |
| **Wirkmechanismus vs. Ritalin** | Releasing Agent (aktive Monoaminfreisetzung) statt Reuptake-Hemmung. Zusätzliche schwache SERT-Aktivität (Serotoninfreisetzung). Glattes Freisetzungsprofil durch Prodrug-Hydrolyse in Erythrozyten. |
| **Wahrnehmungsqualität**        | Konzentrierter, nicht müde, wenig abgelenkt. Diffusität geringer als bei Ritalin. Phosphene/visuelle Wellen bewusster wahrnehmbar.                                                                   |
| **Schlaf-Wach-Übergang**        | Powernap unter Substanz möglich (bei Neurotypischen pharmakologisch ungewöhnlich). Aufwachen erleichtert, prolongierte Atonie fehlend.                                                               |
| **Kardiovaskulär**              | Blutdruck mäßig erhöht unter Metoprolol. Puls stabil, Sättigung optimal.                                                                                                                             |
| **Rebound**                     | Kein psychotischer Rebound beobachtet – entscheidender Unterschied zu Methylphenidat.                                                                                                                |
| **Sensorische Dissoziation**    | Sensorische Hypersensitivität (Phono-/Photophobie) bei Gartenarbeit ohne nachfolgende Migräne. Kompensatorisches Verhalten (Tunnelblick, Basecap) trat auf, Entzündungskaskade blieb aus. Erster beobachteter Vergleichspunkt: gleiche subjektive sensorische Belastung, aber anderes kortikales Ergebnis — belegt die Dissoziation zwischen thalamischem Gating und basaler sensorischer Schwelle (vgl. Abschnitt 3.4.1). |


### **7.3 Therapeutisches Fenster**

Die Dopamin-Hypersensitivität (Migräneliteratur: D2-Rezeptor-Hypersensitivität \+ thalamische Gating-Insuffizienz als Verstärker) bedeutet, dass therapeutischer Effekt bei Mikrodosen erreicht wird, weit unterhalb der ADHS-Standarddosierung:

| Substanz | Therapeutisches Fenster |
| :---- | :---- |
| **Neurotypische ADHS-Dosis** | Lisdexamfetamin 30–70 mg (≈10–22 mg aktives Dexamfetamin) |
| **Vorliegendes Profil** | Lisdexamfetamin 7,5–15 mg (≈2,5–5 mg aktives Dexamfetamin) |
| **Ziel** | Minimale Dosis, die Raphe-Aktivität stabilisiert ohne thalamisches Gating merklich zu verschieben oder sympathisches System zu überreizen |

**Interaktion mit Metoprolol:** Metoprolol blockiert selektiv β1/β2. Lisdexamfetamin aktiviert das gesamte sympathische System inklusive α-Rezeptoren und zentrale Noradrenalinbahnen, die vom Betablocker nicht erfasst werden. Partiell blockiertes System: kardiovaskuläres Risiko reduziert, zentraler Effekt bleibt erhalten. Metoprolol-Reduktion von 100 auf 50 mg klinisch vertretbar bei paralleler Lisdexamfetamin-Gabe.
